# Математическая игра "Между нами девочками"

## Установка
```
npm install
```

### Сборка для отладки
```
npm run serve
```

### Сборка для публикации
```
npm run build
```
## Доступ к БД для статистики

Переименовать api/bd_conf-example.php в api/bd_conf.php.
Создать свою базу данных, исправить учетные данные на свои.
